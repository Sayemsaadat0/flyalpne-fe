## How to use

To begin

```sh
npm install && npm dlx husky install
```

To run development server (PORT = 3000)

```sh
npm run dev
```

To format and test code

```sh
npm run test-all
```

To build files

```sh
npm run build
```

To run production server (PORT = 3001)

```sh
npm run start
```
