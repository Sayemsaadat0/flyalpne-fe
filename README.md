## How to use

To begin

```sh
pnpm install && pnpm dlx husky install
```

To run development server (PORT = 3000)

```sh
pnpm run dev
```

To format and test code

```sh
pnpm run test-all
```

To build files

```sh
pnpm run build
```

To run production server (PORT = 3001)

```sh
pnpm run start
```
